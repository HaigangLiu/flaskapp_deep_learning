A deep learning app for CXR reading

This deep learning app is designed to streamline the Chest X-ray reading process. To get diagnosis based on Chest X-ray (CXR), users can either upload a single image or a zip file of multiple images. In the former use case, we will not only give a numerical probability with state-of-the-art accuracy1, but also a heat map indicates where the pathology is likely to be2.

On the other hand, by leveraging the power of GPU3, we can process a large amount of images rather efficiently. User can compress multiple images into a zip file and we will take it from there. Unlike the single image case, user is required to submit a file (csv or txt) recording the ground truth so that we can perform in-depth analysis.4.

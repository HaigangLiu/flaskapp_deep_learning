Project Overview

This deep learning project aims to use state-of-the-art machine learning algorithm to help lung disease detection(bacterial pneumonia in particular) and thus accelerate the development of vaccine. Several algorithms have been tested and two of them are supported in this app under the hood: ResNet151 and DenseNet 121.

In the meantime, we wish more developers will be involved in developing deep learning projects and application. In this spirit, the following section will be devoted as a tutorial compiliation to walk interested users through the implementation of this project. This tutorial series will be mostly leveraging two modern Python frameworks, Pytorch (used as the backend to train and test models) and Flask (used as the front end for web application). Any contributions are welcomed.
